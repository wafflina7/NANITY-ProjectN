# NANITY-Project N
Nanity is an AI integrated into the computer. It uses local features to ensure maximum privacy and customizability. It is also upgraded and uses "V" to number its models.
It IS being developed continiously and offers alot of features for perfect daily life use.
(Current) V3 model uses local Web browser, a local AI and a red and black clean UI.
UPCOMING v4 model uses rite,features of v3, openCV and other built in local features to add a character which can move around in real time, open apps and comment about the screen BUT the model is currently facing alot of issues.
